## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
\`\`\`bash
# Clone the repository
git clone https://github.com/junaidreal4-hub/N88E-Build-Website.git

# Navigate to directory
cd N88E-Build-Website

# Install dependencies
npm install

# Start development server
npm run dev
\`\`\`

## Build
\`\`\`bash
npm run build
\`\`\`
